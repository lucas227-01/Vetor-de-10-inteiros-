# Vetor-de-10-inteiros-
Projeto em C para treinar lógica de programação utilizando vetores e estruturas de repetição. O programa cria um vetor de 10 números inteiros, recebe valores do usuário e percorre o vetor para encontrar e mostrar o maior número armazenado. Exercício feito para prática durante os estudos de Sistemas de Informação.
